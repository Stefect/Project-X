#  Project-X - Твій Розумний Браузер

> Уявіть собі звичайний браузер, але з суперсилою AI! 

Це **не просто браузер** - це твій особистий AI асистент, який допомагає читати, програмувати, шукати інформацію та навіть підказує, що ти хочеш написати! Зроблений з  для тих, хто цінує свій час.

---

##  Що нового? (v2.2.0 - 08.02.2026)

###  AI-Автозаповнення - GitHub Copilot для всього інтернету!
Найкрутіша фіча! Ти пишеш - AI підказує що далі. Як у смартфоні, тільки в 1000 разів розумніший! 
- Починаєш писати коментар на YouTube
- Чекаєш пів секунди
- З'являється **сіра підказка** від AI: "...це відео дуже корисне!"
- Тисне **Tab** - і текст вставляється сам!
- Працює **скрізь**: Gmail, Facebook, Twitter, форуми, чати

**Приклад:**
```
Ти пишеш: "Слава..."
AI підказує: "...Україні!" 
```

Це не просто словник - це повноцінний AI який розуміє контекст!

###  Розумний Пошук - Шукай як говориш!
Забудь про точні слова! Просто запитай **"де про доставку?"** і браузер знайде все про відправку, навіть якщо це слово не згадується. Це як Google, але набагато розумніший! 
- Натисни **F3** і говори своїми словами
- AI зрозуміє що ти маєш на увазі
- Працює навіть з документами та статтями

###  Code Mate - Друг кожного програміста
Стомився гуглити, що робить цей код? Тепер AI все пояснить за секунду! 
- Відкриваєш StackOverflow чи GitHub
- Бачиш кнопку **" Пояснити"** біля кожного коду
- Клікаєш - отримуєш зрозуміле пояснення
- Навіть знаходить баги, які ти міг пропустити!

###  T9 як у старих Nokia - тільки краще!
Пам'ятаєш T9 на кнопкових телефонах? Тут так само, але в 100 разів розумніший! 
- Починаєш писати - з'являються підказки
- Запам'ятовує твої улюблені фрази
- Працює українською, російською, англійською

##  Чому це круто?

### Для всіх:
-  **Звичайний браузер** - все працює як у Chrome, нічого нового вчити
-  **AI завжди поруч** - виділив слово? Бум! Одразу пояснення (швидше ніж ти встигнеш загуглити)
-  **Зберігай важливе** - знайшов щось цікаве? Збережи в конспект, AI потім узагальнить
-  **Приємно користуватись** - все красиве, швидке та без реклами

### Для програмістів:
-  **Code Mate** - твій особистий code reviewer
-  Аналіз коду прямо на GitHub/StackOverflow
-  Знаходить помилки швидше за тебе
-  Пояснює складний код простими словами

### Для тих, хто вчиться:
-  **Розумний пошук** - знаходить відповіді навіть якщо ти не знаєш як правильно запитати
-  Ідеально для читання статей та підручників
-  AI допоможе зрозуміти складні терміни

### Для охочих писати швидше:
-  **T9 підказки** - як автодоповнення в смартфоні, тільки краще
-  Працює скрізь: Google, Facebook, Gmail, форуми
-  Економить купу часу на написанні

##  Як встановити? (Легко!)

**Потрібно:** Node.js і 5 хвилин вільного часу 

### Підтримувані платформи
-  **Windows** (Windows 10/11)
-  **macOS** (Intel & Apple Silicon)
-  **Linux** (Ubuntu, Debian, Arch, Fedora)

### Крок 1: Завантаж код
```bash
git clone https://github.com/Stefect/Project-X.git
cd Project-X
```

### Крок 2: Встанови все необхідне
```bash
npm install
```
* Хвилинку зачекай - воно саме все скачає*

### Крок 3: Отримай безкоштовний AI ключ
1. Йди на https://console.groq.com/keys (це швидко і безкоштовно! )
2. Зареєструйся (якщо ще не маєш акаунт)
3. Скопіюй свій API ключ
4. Відкрий файл `config.js` (в корені проекту) і вставв його сюди:

```javascript
module.exports = {
  GROQ_API_KEY: 'сюди_вставти_свій_ключ'
};
```

### Крок 3.5: Налаштуй Tor (опціонально для анонімності)
Tor забезпечує анонімний доступ до інтернету.

#### Windows
```bash
# Завантаж Tor Expert Bundle: https://www.torproject.org/download/tor/
# Розпакуй і скопіюй tor.exe в bin/tor/
```

#### macOS
```bash
# Через Homebrew (рекомендовано)
brew install tor
cp $(which tor) bin/tor/tor
chmod +x bin/tor/tor
```

#### Linux
```bash
# Ubuntu/Debian
sudo apt-get install tor
cp /usr/bin/tor bin/tor/tor
chmod +x bin/tor/tor

# Arch Linux
sudo pacman -S tor
cp /usr/bin/tor bin/tor/tor
chmod +x bin/tor/tor
```

*Детальніше в [bin/README.md](bin/README.md)*

### Крок 4: Запускай і насолоджуйся! 
```bash
npm start
```

**Готово!** Браузер відкриється і можеш користуватись! 

##  Як цим користуватись?

###  AI Помічник - Твій особистий Google
**Приклад з життя:** Читаєш статтю, бачиш незнайоме слово "квантовий комп'ютер"
1. Просто виділи це слово мишкою
2.  Tada! З'являється вікно з поясненням
3. AI розповість простими словами що це таке

*Працює з будь-яким текстом: статті, новини, Wikipedia, все!*

###  Code Mate - Наче старший колега поруч
**Приклад:** Знайшов на StackOverflow код, але не розумієш що він робить?
1. Відкриваєш StackOverflow (або GitHub, або будь-яку документацію)
2. Бачиш біля кожного коду кнопку **" Пояснити"**
3. Клікаєш - і AI розповість:
   -  Що робить цей код простими словами
   -  Які головні функції використовуються
   -  Де можуть бути помилки
   -  Як можна покращити

*Працює скрізь: StackOverflow, GitHub, Dev.to, GeeksforGeeks - всюди де є код!*

###  Конспект - Збирай важливе
**Для студентів та тих, хто багато читає:**
1. Знайшов щось важливе? Виділи текст
2. Права кнопка миші → **" Додати в конспект"**
3. Накопичилось багато? Натисни **" Узагальнити AI"** - отримаєш коротке резюме всього!

*Все зберігається локально на твоєму комп'ютері - ніхто не бачить крім тебе*

###  Розумний Пошук - Магія!
**Ситуація:** Читаєш довгу статтю про товар і хочеш дізнатись про доставку

 **Звичайний Ctrl+F:** Шукаєш "доставка" - нічого не знайшов (бо там написано "відправка")
 **Наш AI пошук:** Питаєш "де про доставку?" - знаходить все!

**Як користуватись:**
1. Натисни **F3** (або кнопку **** вгорі)
2. Пиши своїми словами що шукаєш:
   - *"скільки це коштує?"* → знайде ціну
   - *"де автор?"* → знайде інформацію про автора
   - *"як це працює?"* → знайде пояснення принципу роботи
3. AI прочитає всю сторінку і покаже найкращу відповідь

**Супер для:** навчання, шопінгу, читання документації 

###  T9 - Як у смартфоні, тільки краще!
**Працює скрізь:** Google, Facebook, Gmail, форуми, коментарі - де завгодно!

**Як користуватись:**
1. Просто починай писати в будь-якому полі
2. Бачиш підказки? Використовуй **↑↓** стрілочки
3. Натисни **Enter** або **Tab** щоб вибрати
4. Чим більше пишеш - тим розумніший він стає! 

**Фішка:** Він запам'ятовує твої часті фрази і слова. Написав щось один раз - наступного разу просто вибери з підказок!

* Хочеш більше деталей? Глянь [T9-README.md](T9-README.md)*

##  На чому це працює?

*Якщо цікаво що під капотом:*

- **Electron** - те саме, що використовує VS Code, Slack, Discord
- **Chromium** - той самий рушій що в Google Chrome (тому все працює ідентично)
- **Groq AI (LLaMA 3.1)** - найшвидший AI у світі (серйозно, швидше GPT-4!)
- **Code Injection** - наша магія, яка додає кнопки та функції на сайти

*Все безкоштовно та open-source! *

##  Для розробників

*Хочеш щось додати або змінити? Тримай карту проекту:*
```
проект/
├── src/                    # Вихідний код
│   ├── main.js            # Головний процес Electron
│   ├── preload.js         # Preload скрипт
│   ├── config.js          # Конфігурація API
│   └── modules/           # Модулі функціональності
│       ├── inject.js      # Скрипт відслідковування виділення
│       ├── translator.js  # Модуль перекладу
│       ├── code-injector.js  # Code Mate функціонал
│       ├── link-xray.js   # Link X-Ray сканування
│       ├── t9-engine.js   # Ядро T9 (словник, алгоритми)
│       └── t9-ui.js       # Інтерфейс T9
├── public/                # Публічні файли
│   ├── index.html         # UI браузера
│   └── settings.html      # Сторінка налаштувань
├── tests/                 # Тестові файли
│   ├── test-page.html     # Тестова сторінка
│   └── t9-test.html       # Тест T9
├── docs/                  # Документація
│   ├── TESTING.md         # Інструкції з тестування
│   ├── T9-TESTING.md      # Тестування T9
│   ├── T9-README.md       # Документація T9
│   └── QUICKSTART.md      # Швидкий старт
├── package.json           # Залежності проекту
├── CHANGELOG.md          # Історія змін
└── README.md             # Цей файл
```
