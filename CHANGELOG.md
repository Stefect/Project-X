#  Changelog

## [v2.1.0] - Smart Search Update - 2026-02-04

###  Нова функція

####  Розумний Пошук (Smart Ctrl+F)
- **Пошук за сенсом**: Замість точного співпадіння слів, AI розуміє контекст запиту
- **Природна мова**: Запитайте "де про доставку?" - знайде навіть якщо написано "відправка товарів"
- **Інтелектуальний аналіз**: 
  - AI читає весь текст сторінки
  - Знаходить найбільш релевантний фрагмент
  - Автоматично підсвічує знайдений текст
- **Зручний інтерфейс**:
  - Кнопка  у навігаційній панелі
  - Гаряча клавіша **F3**
  - ESC для закриття
  - Enter для пошуку

####  Переваги над звичайним Ctrl+F
-  Chrome: Шукаєте "ціна" - знайде тільки "ціна"
-  Наш браузер: Шукаєте "ціна" - знайде "вартість", "скільки коштує", "price"

####  Ідеально для:
- **Навчання**: Швидко знайти відповідь у підручнику
- **Шопінг**: Знайти інформацію про доставку/гарантію
- **Дослідження**: Пошук специфічної інформації в статтях
- **Документація**: Знайти потрібний приклад коду

###  Технічні зміни
- Додано обробник `smart-search` в main.js
- Інтеграція з Groq AI LLaMA 3.3 70B для аналізу контексту
- Використання Chromium findInPage API для підсвічування
- Оптимізація: обробка до 30,000 символів тексту

## [v2.0.0] - Code Mate Update - 2026-02-04

###  Нові функції

####  Code Mate - Браузер для Програмістів
- **Автоматичне розпізнавання коду**: Браузер автоматично знаходить всі блоки коду на веб-сторінках
- **AI кнопки**: Додає красиві кнопки " Пояснити" поруч з кожним блоком коду
- **Розумний аналіз**: 
  - Пояснює, що робить код
  - Описує основні функції/методи
  - Знаходить потенційні помилки
  - Пропонує покращення
- **Підтримка сайтів**:
  -  StackOverflow
  -  GitHub
  -  MDN Web Docs
  -  GeeksforGeeks

  -  Будь-яка документація з кодом

####  Покращення UI
- Градієнтні кнопки з hover ефектом
- Плавна анімація появи popup
- Автоматичне позиціонування відносно блоків коду
